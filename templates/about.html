Vision Sentinel AI

PROFESSIONAL AI SURVEILLANCE ECOSYSTEM

Application Class

This is a High-Performance Open-Vocabulary Object Detection System. Unlike traditional AI that can only see fixed objects, this application uses a Semantic Neural Engine (YOLO-World) that allows users to define new objects to track in real-time without retraining the model. It is designed for laboratory research, classroom monitoring, and intelligent surveillance.

Real-Time Vision

Asynchronous multi-threaded processing ensures a smooth 30+ FPS video stream while performing complex AI reasoning.

Dynamic Vocabulary

Instantly add specialized items like Lab Equipment, Glasses, ID Cards, or custom items via the Command Settings portal.

Data Extraction

Automatically generates intelligence logs and downloadable CSV reports for academic or security documentation.

Tactical UI

High-fidelity glassmorphic dashboard featuring live status badges, vision scanlines, and tactical overlays.

🚀 Deployment Guide

  • 1. Ensure Python 3.10+ is installed on the host machine.
  • 2. Install dependencies using pip install -r requirements.txt.
  • 3. Execute the Master Launcher: START_AI.bat.
  • 4. Access the interface via http://127.0.0.1:5000.
