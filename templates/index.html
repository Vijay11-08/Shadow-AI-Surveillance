Application Focus

This is a High-Performance Open-Vocabulary Object Detection System. Built using a Semantic Neural Engine (YOLO-World), it allows users to track specialized items instantly without retraining. Ideal for academic research and intelligent surveillance.

Key Functionalities

  • Real-Time Asynchronous Vision: Multi-threaded processing for smooth 30+ FPS movement.
  • Dynamic Intelligence: Add new objects (Glasses, ID Cards, etc.) on the fly via Command Settings.
  • Report Generation: Automated CSV intelligence logs for audit and research.
  • Glassmorphic Tactical UI: Professional dark-mode interface with holographic scanlines.

Deployment Guide

To run this project: Ensure Python 3.10+ is installed, run `pip install -r requirements.txt`, and double-click the `START_AI.bat` master launcher.
